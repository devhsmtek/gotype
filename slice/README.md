# gonet/port

Slice Type Module

Usage :

```go
package main

import (
	"fmt"

	hsmslice "github.com/husamettinarabaci/gotype/slice"
)

func main() {
	slc := []int{1, 2, 3, 4, 5, 6, 7, 8, 9}
	slc = hsmslice.Remove(slc, 3)
}
```

## Installation:

```shell
go get "github.com/husamettinarabaci/gotype/slice@latest"
```
